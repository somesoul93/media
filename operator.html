Pendahuluan

Operator merupakan simbol atau karakter yang biasa dilibatkan dalam program untuk melakukan suatu operasi maupun manipulasi.

Contohnya : Penjumlahan, pengurangan, pembagian, dan lain-lain.

Contoh penggunaan operator :


A + B

Symbol + adalah operator, sedangkan A dan B disebut dengan operand. Karena operator + di atas melibatkan 2 operand, maka operator ini memiliki sifat binary. Ada tiga sifat operator, antara lain :

  • Unary

    Operator yang hanya melibatkan sebuah operand, misalnya : 3++

  • Binary

    Operator yang melibatkan dua operand, misalnya : 4+2

    • Ternary

    Operator yang melibatkan tiga buah operand, misalnya : (a>b) ? a : b

      Operator Aritmatika

      Operator aritmatika adalah operator yang berfungsi untuk melakukan operasi aritmatika atau matematis.

      Berikut berbagai contoh operator aritmatika :

      Operator Keterangan Contoh
      * Perkalian A * B
      / Pembagian A / B
      % Sisa Bagi (Modulus) A % B
      + Penjumlahan A + B
      - Pengurangan A - B
      = Penugasan A = 5

      Operator penugasan (=) berfungsi untuk mengisi atau mengubah isi suatu variable. Operasi A=5 berarti variable A diisi dengan 5.

      Berikut contoh penggunaan Operator Aritmatika :

      
  #include <iostream.h> 
  #include <conio.h> 
  void main() { 
    cout << "Masukan Nilai A : ";  
    cin >> a; 
    cout << "Masukan Nilai B : ";  
    cin >> b: 
    cout << a << " + " << b << " = " << a+b << endl; 
    cout << a << " - " << B << " = " << a-b << endl; 
    cout << a << " * " << b << " = " << a*b << endl; 
    cout << A << " / " << b << " = " << a/b << endl; 
    cout << a << " % " << b << " = " << a%b << endl: 
    getch(); 
  }
      Operator Increment & Decrement

      Operator Increment dan Decrement berfungsi untuk menaikkan atau menurunkan suatu nilai bilangan bulat. Terdapat dua operator, yaitu :

      Operator Keterangan
      ++ Operator penaikan
      -- Operator Penurunan

      Operasi :

      
  A = A+1;
  B = B-1;

      Dapat disederhanakan menjadi :

      
  A++;
  B++;

      Atau :

      
  ++A;
  ++B;

      Kedua bentuk penulisan di atas memiliki fungsi yang berbeda, yaitu :

      • Jika diletakkan di depan variabel. Maka proses penambahan atau pengurangan akan dilakukan sesaat sebelum atau langsung pada saat menjumpai ekspresi ini sehingga nilai variabel tadi akan langsung berubah begitu ekspresi ini di temukan.

      • Jika diletakkan di belakang variabel. Maka proses penambahan atau pengurangan akan dilakukan setelah ekspresi ini di jumpai atau nilai variabel akan tetap pada saat ekspresi ini ditemukan.

      Berikut contoh penggunaannya :

      
  #include <stdio.h> 
  #include <conio.h> 
  void main() 
  { 
     int a = 8, b = 4; 
     printf(" Nilai A   = %d", a); 
     printf("\n Nilai A++ = %d", a++); 
     printf("\n Nilai A   = %d", a); 
     printf("\n Nilai B-- = %d", B--): 
     printf("\n Nilai B   = %d", b); 
     getch(); 
  }
      Operator Relasi (Perbandingan)

      Operator relasi ini digunakan untuk membandingkan dua buah nilai. Hasil dari perbandingan operator ini menghasilkan nilai numeric 1 (True) atau 0 (False). Berikut adalah operasi Relasi :

      Operator Keterangan
      == Sama dengan (bukan penugasan)
      != Tidak sama dengan
      > Lebih dari
      < Kurang dari
      >= Lebih dari atau sama dengan
      <= Kurang dari atau sama dengan

      Contoh penggunaan operasi relasi :

      
  // Penggunaan operator relasi dan if, if else 
  #include <iostream.h> 
  void main() 
  { 
     int m = 5, n = 7; 
     if (m == n) cout << M << " Sama Dengan " << n << endl; 
     else if (m != n) cout << m << " Tidak Sama Dengan  " << n << endl; 
     else if (m > n) cout << m << " Lebih Besar Dari " << n << endl; 
     else if (m < n) cout << m << " Lebih Kecil Dari " << n << endl; 
  }

      ! NOT !X
