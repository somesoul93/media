Identifier

Identifier atau pengenal adalah suatu nama yang biasa dipakai dalam pemrograman untuk menyatakan :

  1. Variable
  2. Konstanta
  3. Tipe data
  4. Fungsi
  5. Label
  6. Obyek

Identifier bias disebut juga nama yang diciptakan oleh programmer.

Penamaan identifier terdiri atas satu atau beberapa karakter huruf, angka, dan/atau garis bawah(_), serta harus diawali dengan huruf atau garis bawah(_)

Reserved word

Reserved word atau juga bisa disebut keyword, adalah identifier system yang mempunyai makna khusus bagi compiler. Karena itu keyword tidak dapat digunakan sebagai identifier. Berikut adalah reserved word pada C++ :

Asm Auto Break Case Char class Const Continue Default Delete Do Double Else Enum Extern
Float For Friend Goto If Inline Int Long New Operator Private Protected Public Register Return
Sort Signed Sizeof Static Struct Switch Template This Typedef Union Unsigned Virtual Void While
Tipe data

Tipe data pada C++ meliputi :

  • char
  • int
  • long
  • float
  • double
  • long double

tipe data yang berhubungan dengan bilangan bulat adalah char, int, dan long. Sedangkan lainnya berhubungan dengan bilangan pecahan.

Berikut adalah ukuran berbagai tipe data dasar :

Tipe Data Ukuran Memori Jangkauan (range) Jumlah digit presisi Format
Char 1 byte -128 hingga 127-%c
Int 2 byte -32768 hingga 32767-%i,%d
Long 4 byte -2.147.438.648 hingga 2.147.438.647-
Float 4 byte 3.4x10-38 hingga 3.4x10386-7%f
Double 8 byte 1.7x10-308 hingga 3.4x1030815-16%lf
Long double 10 byte 3.4x10-4932 hingga 1.1x10483219
Variabel & Konstanta

Variabel bisa dikatakan sebagai wadah yang memiliki tipe data yang berfungsi untuk menampung suatu nilai. Nilai yang terdapat dalam variable dapat diubah sesuai kebutuhan selama program berjalan.

Untuk mendeklarasikan atau mengenalkan sebuah variable, dilakukan dengan format :


  tipe_data nama_variabel

contoh pendeklarasian variable :


  int jumlah;
  float harga,total;

penugasan atau pengisian nilai variable, dapat dilakukan dengan menggunakan operator penugasan (=)

contoh pemberian nilai 10 ke variable jumlah :


  jumlah = 10;

Konstanta adalah juga bisa dikatakan sebagai wadah, namun nilainya tidak difungsikan untuk dapat diubah.
