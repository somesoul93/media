Pendahuluan

Untuk memanipulasi perulangan, terdapat beberapa pernyataan yang dapat digunakan. Tiap tiap pernyataan memiliki karakteristik masing-masing. Berikut penjelasannya :

Pernyataan while

While adalah salah saru pernyataan yang berfungsi untuk memproses suatu pernyataan atau beberapa pernyataan beberapa kali. Bentuk umumnya :


    while(ungkapan){  
        pernyataan1;
        pernyataan2;
    }

Pernyataan1 dan pernyataan2 akan dieksekusi selama ungkapan bernilai benar. Sehingga pernyataan while dapat digambarkan pada diagram alir berikut :

Berikut adalah contoh penggunaan while dalam program :


    // cpp10x.cpp
    // Contoh pemakaian while
    // untuk menampilkan tulisan sebanyak 10 kali

    #include <iostream.h>

    void main(){
        int i;                 //Variabel penghitung

        i = 0;                 //mula-mula isi dengan nol
        while(i < 10){
            cout << "C++" << endl;
            i++;               //naikkan hitungan 1 kali
        }
    }
Pernyataan Do-While

Sedikit berbeda dengan pernyataan while, pernyataan do-while akan mengeksekusi pernyataan minimal 1 kali saat ungkapan bernilai false. Bentuk umum dari do while adalah :


    do{
        Pernyataan1;
        Pernyataan2;
    }while(ungkapan);

Pernyataan 1 dan pernyataan 2 dieksekusi sampai ungkapan bernilai salah. Berikut diagram alir pernyataan do-while :

Berikut contoh penggunaan do while :


    // cpp10x2.cpp
    // Contoh pemakaian do=while
    // untuk menampilkan tulisan sebanyak 10 kali

    #include <iostream.h>

    void main(){
        int i;              //Variabel penghitung

        i = 0;              //mula -mula isi dengan nol
        do{
            cout << "C++" << endl;
            i++;            //naikkan hitungan 1 kali
        }while(i < 10);
    }
Pernyataan for

Pernyataan for juga berguna untuk mengulak pengeksekusian terhadap satu atau sejumlah pernyataan. Bentuk umum pernyataan for adalah sebagai berikut : 


    for(ungkapan1;ungkapan2;ungkapan3){
        pernyataan;
    }

Pernyataan di atas identik dengan :


    ungkapan1;
    while(ungkapan2){
        pernyataan;
    ungkapan3;
    }

Jadi :

  • Ungkapan1 adalah pernyataan inisialisasi sebelum masuk ke while.
  • Ungkapan2 berlaku sebagai kondisi yang menentukan perulangan dilanjutkan atau tidak
  • Ungkapan3 digunakan sebagai pengatur variable yang digunakan dalam ungkapan1

Berikut adalah contoh penggunaan for :


    // cpp10x3.cpp
    // Contoh pemakaian for
    // untuk menampilkan tulisan sebanyak 10 kali

    #include <iostream.h>

    void main(){
        int i;          //Variabel penghitung

        for(i=0 ; i < 10 ; i++){
            cout << "C++" << endl;
        }
    }
