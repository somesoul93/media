Apa itu Program?

Program adalah sebuah barisan instruksi (biasa disebut programming statements), yang dieksekusi satu demi satu, yang biasanya secara berurutan, seperti yag terlihat pada bagan di atas.

Contoh : program dibawah ini (Lingkaran.cpp) akan menghasilkan keluaran berupa luas dan keliling lingkaran. Perhatikan, setiap instruksi dieksekusi secara berurutan satu demi satu sesuai dengan baris program.


    #include <iostream.h>
    #include <stdio.h>

    int main(){
      double jari, keliling,luas;
      double pi = 3.14159265;

      cout << "Masukkan Jari-Jari: ";
      cin >> jari; //baca inputan keyboard

      //Hitung Luas dan Keliling
      luas = jari * jari * pi;
      keliling = 2.0 * jari * pi;

      //Tampilkan hasil
      cout << "Panjang Jari-Jari " << jari << "." << endl;
      cout << "Luas " << luas << "." << endl;
      cout << "Keliling " << keliling << "." << endl;
      return 0;
    }

Keluaran Program :


    Masukkan Jari-Jari: 1.2
    Panjang Jari-Jari 1.2.
    Luas 4.52389.
    Keliling 7.53982.

Penjelasan program :


    #include <iostream.h>
    #include <stdio.h>

Baris ini bukanlah suatu pernyataan atau statement. Baris tersebut menginstruksikan kepada compiler untuk menyisipkan file lain (dalam hal ini file iostream.h dan stdio.h) saat profram dikompilasi. File dengan ekstensi .h disebut juga file header yang berisi berbagai deklarasi seperti fungsi, variable, dan sebagainya.


    double jari, keliling, luas;
    double pi = 3.14159265;

kita mendeklarasikan tiga buah variable dengan tipe data double, yakni jari, keliling, dan luas. Tipe data double tidak sama dengan int, double dapat menyimpan nilai decimal (floating-point number).

Kita juga mendeklarasikan variable double bernama pi, dan diinisialisasi nilainya, yakni 3.14159265


    cout << "Masukkan Jari-Jari: ";
    cin >> jari;

kita menggunakan cout << untuk menampilkan pesan, dan cin >> untuk membaca inputan user ke dalam variable jari.


    luas = jari * jari * pi;
    keliling = 2.0 * jari * pi;

melakukan penghitungan


    cout << "Panjang Jari-Jari " << jari << "." << endl;
    cout << "Luas " << luas << "." << endl;
    cout << "Keliling " << keliling << "." << endl;

menampilkan hasil.

Perhatikan!

Baris kode yang ada di dalam main() akan dieksekusi satu persatu secara berurutan.

Komentar

Komentar adalah hal yang penting dalam program. Komentar adalah bagian yang tidak akan dieksekusi oleh program. Kamu akan membutuhkan program untuk memberikan penjelasan pada bagian tertentu seperti :

  • Tujuan/fungsi program
  • Kredit programmer untuk menuliskan revisi
  • Keterangan-keterangan lain tentang keguanaan sejumlah pernyataan dalam program.

Komentar juga berfungsi agar kode program dapat dipahami oleh programmer lain.

Cara penulisan komentar ada dua jenis :

  1. Single line

    Komentar diawali dengan dua garis miring (//). Semua tulisan yang terletak setelah tanda // hingga akhir baris tidak akan dieksekusi. Dengan kata lain komentar dengan // berlaku untuk satu baris

    Contoh :

    #include <iostream.h>  // menyertakan file iostream.h

    Pada contoh di atas, 

    // menyertakan file iostream.h

    tidak akan dieksekusi.

  2. Multiline

    Selain menggunakan //, komentar pada c++ dapat ditulis dalam bentuk :

     /* komentar */

    Komentar diawali dengan tanda /* dan diakhiri dengan */. Sepasang karakter tersebut bias diletakkan di baris yang berbeda, sehingga memungkinkan untuk membuat komentar beberapa baris :

    
    /*  ini 
      Adalah contoh
      Multiline comment c++
    */
